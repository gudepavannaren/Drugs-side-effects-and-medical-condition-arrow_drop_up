## Project Overview

This project analyzes a dataset containing information about various drugs, their side effects, and the medical conditions they treat. The dataset was sourced from **drugs.com** and contains details on **2,931 drugs** across **47 medical conditions**.

The project covers end-to-end data science workflow — from raw data cleaning to exploratory data analysis (EDA) and a machine learning classification model.


## Objectives

- Clean and preprocess real-world pharmaceutical data
- Explore patterns between drugs, side effects, and medical conditions
- Visualize key insights using charts and graphs
- Build a Machine Learning model to classify drug rating categories


## Dataset Description

| Column | Description |
 `drug_name` | Name of the drug |
 `medical_condition` | Condition the drug is used to treat |
 `side_effects` | Listed side effects of the drug |
 `generic_name` | Chemical/generic name of the drug |
 `drug_classes` | Class the drug belongs to |
 `brand_names` | Brand names under which drug is sold |
 `activity` | Relative site visitor activity (popularity) |
 `rx_otc` | Prescription (Rx) or Over-the-Counter (OTC) |
 `pregnancy_category` | FDA pregnancy risk category (A/B/C/D/X/N) |
 `csa` | Controlled Substances Act schedule |
 `alcohol` | Whether the drug interacts with alcohol |
 `related_drugs` | Other related drugs |
 `medical_condition_description` | Description of the medical condition |
 `rating` | User rating (1 = not effective, 10 = most effective) |
 `no_of_reviews` | Number of user reviews |
 `drug_link` | URL to drug info page |
`medical_condition_url` | URL to condition info page |

**Dataset Size:** 2,931 rows × 17 columns
